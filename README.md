# Google Maps Project

פרויקט שמשלב את Google Maps API ליצירת אפליקציה לניהול מקומות.

## 💡 מה הפרויקט עושה:
- הצגת מפת גוגל
- הוספת מיקומים מועדפים
- עריכת הגדרות משתמש

## 🛠 טכנולוגיות:
- HTML + CSS + JavaScript
- Google Maps API

## 🧭 איך להריץ:
1. לשכפל את הריפו (`git clone`)
2. לפתוח את הקובץ `index.html` בדפדפן
3. לוודא שיש מפתח API חוקי (Google Maps)
